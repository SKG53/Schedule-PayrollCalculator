How to use:
  1. Make sure each entity has a schedule loaded in the Schedules tab first.
  2. For each entity below, drop in physical timecard photos and/or EasyClocking screenshots.
  3. Click Run OCR. The app extracts clock-in/out data for every punch.
  4. Review each row. Any row with a flag needs your eyes. Fix names, dates, or times in place.
  5. Set the mandatory break minutes per entity. Then go to the Payroll tab.
How to use:
  1. Upload last week's schedule .xlsx — or click start blank in the drop zone to enter shifts manually.
  2. Edit shifts using formats like 6:45AM - 2:15PM or OFF. Overnight shifts are supported (11PM - 7AM).
  3. Set the new week's start date — the date headers update automatically.
  4. Export either the employee-facing schedule or the manager view with hours totals.
  5. Use the entity tabs below to manage multiple entities — each has its own schedule.
