Schedule Planning & Payroll Calculation

How to use:
  1. Load each entity's schedule first in the Schedules tab.
  2. For each entity, drop in physical timecard photos and/or EasyClocking screenshots, then click Run OCR.
  3. Rows are split into one sub-tab per employee. Review each employee: correct names, dates, or times in place, then approve the row.
  4. Need to tweak a row you already approved? Click Unapprove to unlock it, edit, then reapprove.
  5. Head to the Payroll tab to set wages and export.
How to use:
  1. Give this entity a name (required before you can move to the next tab), then upload last week's schedule .xlsx, or click start blank to enter shifts by hand.
  2. Edit shifts using formats like 6:45AM - 2:15PM or OFF. Overnight shifts are supported (11PM - 7AM).
  3. Set the new week's start date — the date headers update automatically. The calendar below shows staffing counts per hour.
  4. Export the employee-facing schedule or the manager view with hours totals. Use the entity tabs above to manage multiple entities.
